## Input files format

To use the parser in the main file, the format should follow this form:

# int numpoints
# list points
# double length1
# double radius1
# double epsilon
# double area1
# int Q_PTS
# int quads
# double dist
# int N_PTS
# int K_PTS
# int A_PTS
# int B_PTS
# int C_PTS
# int D_PTS
# int E_PTS
# int F_PTS
# int G_PTS
# double length2
# double radius2
# double area2
# list LCM
# list PUV
